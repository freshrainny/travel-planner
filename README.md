# ✈️ AI 旅行规划师

> 一款基于 AI 的智能旅行行程规划工具，支持自定义 AI 模型、地图可视化、景点攻略、行程分享等功能。

🌐 **在线体验：[https://freshrainny.github.io/travel-planner/](https://freshrainny.github.io/travel-planner/)**

---

## 🚀 快速开始

1. 打开网站，点击右上角 **⚙️ 设置** 填入你的 AI API Key
2. 在左侧表单填写 **目的地**、**出发日期**、**天数**，加上旅行偏好
3. 点击 **✨ 开始规划**，AI 自动生成完整行程
4. 点击每天的行程卡片展开详情，查看景点介绍、交通方式
5. 自由拖拽调整顺序，或手动编辑/添加/删除景点

---

## ✨ 核心功能

### 🤖 多模型 AI 支持
- 支持 **Claude**（Anthropic）、**GPT / GPT-5**（OpenAI）、**DeepSeek** 等主流模型
- 可自定义 API Endpoint，兼容任意 OpenAI 格式的接口
- 智能两阶段生成：先生成行程框架，再按天加载详细内容，避免 Token 超限

### 🗺️ 地图可视化
- 所有景点自动标注在交互地图上，显示游览顺序编号
- 支持按天筛选，切换查看每天的路线分布
- 住宿地点用绿色图标单独标注
- 使用英文地图底图，清晰易读

### 🔍 景点 AI 攻略
点击任意景点的 🔍 按钮，弹出 AI 实时生成的四合一攻略：

| 标签 | 内容 |
|------|------|
| 📋 概览 | 景点介绍、亮点、最佳游览时间、建议时长、人流情况 |
| 🎫 门票与时间 | 票价、购票方式、开放时间、闭馆日、免票政策 |
| ⚠️ 注意事项 | 4+ 条实用提醒（着装、预约、排队、禁带物品等） |
| 🍜 周边美食 | 3-5 家周边餐厅推荐，含距离、人均消费、特色标签 |

### ✏️ 自由编辑行程
- **拖拽排序**：任意调整景点顺序
- **编辑景点**：修改名称、类型、时间、备注
- **添加景点**：向任意一天插入新活动
- **删除景点**：一键移除不需要的安排
- **AI 重新规划**：保留现有景点，让 AI 重新优化某一天的行程

### 🏨 住宿管理
- 为每一天设置住宿酒店/民宿名称
- 住宿信息显示在天卡片头部和地图上

### 🔗 一键分享行程
- 点击顶部 **分享** 按钮，自动生成含完整行程数据的分享链接
- 朋友打开链接即可查看完整行程，**无需登录，无需后端**

## 📱 使用技巧

- **行程太长加载慢？** 正常，AI 按天分批加载，展开哪天才会请求 AI，不会一次性消耗大量 Token
- **生成内容不满意？** 点击某天的 **🤖 重新规划** 按钮，可以附加要求（如"多加一些文化体验"）让 AI 重新生成
- **攻略信息不准确？** AI 攻略基于训练数据生成，票价/开放时间请以景区官方公告为准，点击🔄重新生成可获得不同版本

## 🧰 技术栈

- **前端**：纯 HTML + CSS + Vanilla JS，无需构建工具，单文件部署
- **地图**：[Leaflet.js](https://leafletjs.com/) + CartoDB Voyager 底图（英文标注）
- **拖拽**：[SortableJS](https://sortablejs.com/)
- **AI**：统一 `callAI` 函数，兼容 Anthropic 和 OpenAI 格式 API
- **分享**：URL Hash + Base64 编码行程数据，纯前端实现
